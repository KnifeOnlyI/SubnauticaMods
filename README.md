# KnifeOnlyI's Subnautica mods

The complete list of all subnautica mods created by me.

## For developers

### Requirements

- .NET Framework 4.8.1
- Environment variable `STEAM_LIBRARY` set to your steam library path (`<steam_folder>/steamapps/common`)

Feel free to create a new merge requests.
